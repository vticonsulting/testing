
## Mocking Axios in Jest + Testing Async Functions

How to test asynchronous functions with Jest and it's built-in mocking capabilities 

```js
// ./src/services/Unsplash
``

```js
// ./src/__mocks__/axios.js
export default {
  get: jest.fn(() => Promise.resolve({
    data: {},
  })),
}
```
