## Standards

### No scoped styles

Instead, give the component's base element a class and use that instead.

```html
<!-- bad -->
<style scoped>
.component {}
</style>

<!-- // good -->
<template>
  <div class="component">
  </div>
</template>

<style>
.component {}
</style>
```
