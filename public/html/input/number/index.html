<input type="number">

Example

Usage

<input
  type="number"
  id="number"
  min="0"
  max="120"
  step="1"
  required
/>

Description

The number type especially useful when requiring a number input. Supported browsers will not allow any non-numeric value through.

min and max attributes stop values outside of the required bounds from being submitted. The step attribute requires the number is a multiple of the supplied value. This also effects the interface, which will then only step up or down those increments.
