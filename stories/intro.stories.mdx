## Animation

To maintain performance in our animation timeline, we want to try
our best to limit our animated properties to a **core four**.

```scss
// Scale
.scale-150 {transform: scale(1.5)}
// Position
.translate-x-full {transform: translateX(100%)}
// Rotation
.rotate-180 {transform: rotate(180deg)}
// Opacity
.opacity-50 {opacity: 0.5}
```
